⚡ Energy Data  Dashboard

 “Cloud-based Energy Analytics Dashboard leveraging ASP.NET Core, Azure messaging, and Power BI for data visualization

This project follows an end-to-end data pipeline:
🔄 Project Flow
Excel → API → Messaging Service → SQL Database → Power BI

📊 Excel: Raw generation data source

⚙️ API: Processes and pushes data

🔁 Messaging: Handles data flow (queue-based)

🗄️ SQL: Stores structured data

📈 Power BI: Visualizes KPI, deviation & trends
